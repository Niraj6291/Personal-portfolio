# 💼 Personal Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, projects, resume, and contact information in a clean, responsive, and professional layout. It is designed to reflect my identity as a developer and provide an interactive way for visitors to know more about me.

## 🔧 Technologies Used

- HTML5
- CSS3
- JavaScript (if applicable)
- Responsive Design

## 📂 Features

- Personal introduction and bio
- Display of technical skills and tools
- Portfolio section with project highlights
- Resume download option
- Contact information with social media links
